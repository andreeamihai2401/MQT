1. Create a topic having 3 brokers as --bootstrap-server. 
The brokers' ports are taken from the docker-compose_kafka.yml
The topic should have 4 partitions and replication-factor 2
The name of the topic is events2.  
Purge the topic. Set retention to a small value

/usr/bin/kafka-topics --create --bootstrap-server kafka:9092,kafka2:9093,kafka3:9094 --partitions 4 --replication-factor 2 --topic events2

/usr/bin/kafka-configs --alter --bootstrap-server kafka:9092,kafka2:9093,kafka3:9094 -entity-name events2 -entity-type topics --add-config retention.ms=10

2. Check the topic. Create a producer and send some messages.

After sometime when you run this command the messages will not be shown

/usr/bin/kafka-console-producer --bootstrap-server kafka:9092,kafka2:9093,kafka3:9094 --topic events2 

3. Set retention to normal value like retention.ms=604800000
 Check the topic. Create a producer and send some messages.

/usr/bin/kafka-configs --alter --bootstrap-server kafka:9092,kafka2:9093,kafka3:9094 -entity-name events2 -entity-type topics --add-config retention.ms=604800000

/usr/bin/kafka-console-producer --bootstrap-server kafka:9092,kafka2:9093,kafka3:9094 --topic events2 

4. Delete topic

/usr/bin/kafka-topics --delete --bootstrap-server kafka:9092,kafka2:9093,kafka3:9094 --topic events2
